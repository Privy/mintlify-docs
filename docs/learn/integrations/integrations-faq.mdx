**Privy doesn't integrate with my e-commerce site platform. Can I still use Privy?**

*Yes, you can manually install Privy on your website by following these instructions.*

**I'm seeing an 'Unauthorized' error with one of my integrations. What do I do?**

*'Unauthorized' errors can occasionally happen, for a number of reasons. Learn more about the potential causes and how to resolve these errors with* [*this guide*](https://privy.mintlify.app/docs/learn/integrations/resolve-an-unauthorized-error)*.*

**Does Privy work with single-page applications (SPA)?**

*Yes! Learn more about our compatibility, advantages, and disadvantages in* [*this resource*](https://privy.mintlify.app/docs/learn/integrations/does-privy-work-with-spa#does-privy-work-with-spas)*.*

**I'm using a third-party email service provider (ESP). Can I sync coupon codes from Privy sign ups to my ESP?**

*Yes,* [*follow this guide*](https://privy.mintlify.app/docs/learn/integrations/sync-coupons-to-esp) *for best practices on syncing coupon codes to your ESP as new contacts sign up.*
